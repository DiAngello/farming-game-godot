<!-- ABOUT THE PROJECT -->
## Sobre o projeto 💡

![Demo]()

Este projeto é um **jogo de fazenda (Farming Game)** desenvolvido utilizando a engine **Godot**.  

O objetivo é aprender conceitos de desenvolvimento de jogos como:

- movimentação de personagem;
- interação com o ambiente;  
- organização de cenas e scripts;  
- uso de tilesets e assets;

<!-- GETTING STARTED -->
## Como iniciar ▶️

### Pré-requisitos ✅

Antes de começar você precisa ter instalado:

- Godot Engine (4.x recomendado)
- Git

Download da Godot:  
https://godotengine.org/download

### Instalação 📦

1. Clone o repositório:

```sh
git clone https://github.com/DiAngello/farming-game-godot.git
   ```

Ou baixar o arquivo ZIP do repositório no GitHub.

2. Abra a pasta do jogo exportado
   
4. Execute o arquivo:
   
```sh
FarmingGame.exe
```
